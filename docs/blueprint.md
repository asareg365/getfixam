# **App Name**: FixAm Ghana

## Core Features:

- Provider Listing: Display a list of local artisans, searchable by category and area (Berekum).
- Contact Information: Enable users to contact artisans directly via phone call or WhatsApp.
- Location Display: Show the area/neighborhood where the artisan is located.
- Ratings and Reviews: Allow users to submit star ratings and written reviews for artisans.
- Category Browsing: Enable users to browse artisans by predefined categories (e.g., Plumber, Electrician).
- Provider Onboarding: Create CMS functionality for admins to manually onboard and verify artisans.
- Data Storage: Store artisan information, reviews, and category details in Firestore.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7849), evoking a sense of energy and reliability, reflective of the app's practical problem-solving nature.
- Background color: Light peach (#F8E7D9), providing a warm and inviting backdrop without distracting from the content.
- Accent color: Yellow-orange (#FFAC33) to highlight key interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear and representative icons for each artisan category. Icon set to be simple and monochromatic.
- Maintain a clean and straightforward layout, emphasizing readability and ease of navigation.
- Incorporate subtle transition animations to enhance user experience.
